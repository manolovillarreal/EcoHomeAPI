EcoHome Store API

Esta API corresponde al desarrollo del backend para un caso académico de e-commerce, enfocado en la gestión de productos y control de acceso mediante autenticación.

Descripción general

La aplicación implementa una arquitectura RESTful utilizando Node.js y PostgreSQL, incorporando persistencia de datos, autenticación mediante JWT y control de roles (admin y cliente).

Funcionalidades principales

  • Registro y autenticación de usuarios
  • Gestión de catálogo de productos (CRUD)
  • Protección de rutas mediante JWT
  • Control de acceso basado en roles

Endpoints principales

POST /auth/signup – Registro de usuario
POST /auth/login – Autenticación
GET /products – Listar productos
GET /products/:id – Obtener producto
POST /products – Crear producto (admin)
PUT /products/:id – Actualizar producto (admin)
DELETE /products/:id – Eliminar producto (admin)

Documentación y pruebas

La API cuenta con un archivo openapi.json que puede ser importado en herramientas como Postman. También se incluye una colección de pruebas para validar el flujo completo de autenticación y CRUD.
